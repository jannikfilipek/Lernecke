Server

Es gibt verschiedene Arten von Servern, grundsätzlich jedoch ist die Idee, dass man von verschiedenen Computern, oder auch Clients genannt, auf Informationen zugreifen kann, welche auf von einem andren Client kommen, ohne direkt auf dieses zugreifen zu müssen.
Die Verbindung von einem Client zu einem Server läuft über ein sogenanntes Handshake Verfahren, welches darauf basiert, dass der Client eine Anfrage sendet, mit einer Zahl. Der Server antwortet auf diese Anfrage, indem er die Zahl nimmt und 1 addiert und selber noch eine Zahl zurückschickt. Nun wiederholt der Client die Aktion von dem Server. Wenn dieser Austausch gelingt, weiß der Client und der Server, dass eine Verbindung hergestellt wurde.
