Mailserver

E-Mails werden auf dem Server zwischengespeichert. Sobald der Nutzer seine E-Mails abruft, wird die E-Mail vom Server gelöscht und ist dann nur noch auf dem jeweiligen Client verfügbar. (POP3)

Ein Datenaustauschanalyse zeigt, dass ein Ablauf aus:

- Verbindung (3-way-handshake)

- Anmeldung

- Senden der E-Mail

- Trennung

besteht

Genauer ausgeführt (Anmeldung, Senden der E-Mai, Trennung)

Client zu Server: HELO (startet die SMPT Sitzung)

Server zu Client: Verbindung aufgebaut

Client zu Server: Mail form..

Server zu Client: Sender ok

Client zu Server: RCPT to.. (Empfänger)

Server zu Client: Empfänger

Client zu Server: DATA (Inhalt der Daten)

Server zu Client: Bereit (nennt das Ende der Daten)

Client zu Server: Sende Daten (hier könnte man die volle E-Mail lesen)

Server zu Client: Mail queued for delivery (bereit zum senden)

Client zu Server: Quit

Server zu Client: Verbindung wird beendet
